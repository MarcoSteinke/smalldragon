SmallDragon Roadmap

SmallDragon is currently Work-in-Progress and there are many things that would make it better. These are Features that may be implemented in the future, along with their priority.

  • Value Range Propagation
  • Array Constants [Medium Priority]
  • Type Parameters/Generics [Medium Priority]
  • Collections (List, Maps, ...) (depends on Generics) [Low Priority]
  • Higher-Order Functions (map, filter, zip, ...) (depends on Collections) [Low Priority]
  • Linear/Affine Types [Low Priority]
  • Tuples [Medium Priority]
