SmallDragon Local Variable Type Inference

SmallDragon can infer the type of a local variable for you, so it is optional to declare the type.

This type inference however, has a runtime cost which rises with the size of the expression.

The type of '3+4' can be quickly computed, the type of '3*(call1(3,3)*4)+7 > call2()' might take longer,
as the entire tree of the expression has to be traversed.

The current implementation uses memoization, so the type of a local variable needs only be computed once.

Example 


fn main () ~>; Int { 
	a = 3;
	return a;
}
