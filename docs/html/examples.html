Example 1: Harshad Numbers

This example presents a basic implementation of the Harshad numbers. A Harshad Number is a number that is divisible by the sum of its digits. For example 16218 is a Harshard number, because the digits sum up to 18 and 16218 mod 18 equals 0, so it is divisible by the sum of its digits. If you do want to know more about the topic, you can click here and find out! So let's start with the implementation:

1. Define the entry point
As in other programming languages, in smalldragon you start with the definition of your entry point. In smalldragon the main method (= entry point) is inspired by the version from the C programming language.

The following code snippet demonstrates the most basic form of the main method in smalldragon: 
		
fn main () ~> int {
	
	return 0;

}
  • fn is used to define a method (function)
  • main is the name of the method
  • () means no parameters for this method
  • ~> TYPE tells the compiler which type is returned


The main method always returns an integer to return a status code to the operating system. In this case smalldragon returns 0 as "everything is okay" (like in C).

2. Implement the logic:
The logic will be in a separate method called testharshad and will have the following properties:
  • parameter: int n
  • return type: int

#include 

fn main () ~> int {

	i = 1;
	loop 20 {
		
		testharshad(i);
		printf("\n");
		i = i + 1;
	}
	
	return 0;
}

fn testharshad (int n) ~> int {

	norig = n;
	sum = 0;
	
	while n != 0 {

		sum = sum + n % 10;
		n = n / 10;
	}

	bool res = norig % sum == 0;

	if res { 
		printf("%d-is-harshad", norig);
	} else {
		printf("%d-is-not-harshad", norig);
	}


	return 0;
}
