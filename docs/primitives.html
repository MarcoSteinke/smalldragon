SmallDragon Primitive Data Types

The primitive Data Types available in SmallDragon are used similar to those available in C, and are a subset of those.

Integer Types

TypeWidth (in bytes)transpiles to:
Int8 8 int8_t
Int16 16 int16_t
Int32 32 int32_t
Int64 64 int64_t

There is also Int, which transpiles directly to 'int' in C. It should not be used if you want a specific size.
