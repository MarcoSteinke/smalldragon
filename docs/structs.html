SmallDragon Composite Data Types

SmallDragon uses 'struct' to define a Composite Data Type.
SmallDragon structs like C structs semantically, but different syntactically. SmallDragon uses ';' at the end of each struct member declaration, same as in C, because this notation is both simple on it's own and also familiar to those who know C.

Example 

	
	struct MyStruct {
		Int a;
		MyStruct b;
	}

Automatically Generated Subroutines

For each declared struct X, sd generates:

  • newX() - allocates memory for an instance of struct X
  • makeX(...) - like newX, but also accepts all members as arguments
  • copyX(X x) - returns a shallow copy
  • delX(X x) - frees (shallow) the instance x
  • printX(X x) - prints the member values

These subroutines are for convenience, so you don't have to write them.
